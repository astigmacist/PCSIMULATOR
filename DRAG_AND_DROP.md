# 🖱️ Drag and Drop Функционал

## Обзор

В симуляторе сборки ПК теперь реализован полноценный drag and drop (перетаскивание) функционал для установки комплектующих.

## Как использовать

### 1. Выбор комплектующих

- Откройте нужную категорию компонентов в левой панели (нажмите на заголовок категории)
- Найдите нужный компонент в списке
- Компоненты, которые можно перетащить, имеют иконку "🖱️ Перетащите →" при наведении

### 2. Перетаскивание

**Шаг 1:** Нажмите и удерживайте левую кнопку мыши на выбранном компоненте
- Курсор изменится на "захват" (grabbing)
- Компонент станет полупрозрачным

**Шаг 2:** Переместите компонент в правую часть экрана на нужный слот
- Слоты подсвечиваются желтым цветом при наведении
- Анимация пульсации показывает, что слот готов принять компонент

**Шаг 3:** Отпустите кнопку мыши над нужным слотом
- Компонент автоматически установится, если он совместим
- При несовместимости появится уведомление с причиной и рекомендациями

### 3. Типы слотов

| Компонент | Куда перетаскивать |
|-----------|-------------------|
| 🏠 Корпус | Специальная зона вверху правой панели |
| 🔌 Материнская плата | Большой слот "Материнская плата" |
| 🧠 Процессор | Маленький слот "CPU" внутри материнской платы |
| 💾 Оперативная память | Слоты "RAM 1-4" |
| 🎮 Видеокарта | Слот "Видеокарта" |
| 💿 Накопители | Слоты "M.2/NVMe" и "SATA 1-2" |
| ⚡ Блок питания | Слот "Блок питания" |

## Визуальные подсказки

### Индикаторы состояния

- **Зеленая рамка** - компонент установлен
- **Желтая рамка с пульсацией** - слот готов принять компонент (при перетаскивании)
- **Красная рамка** - компонент несовместим
- **Полупрозрачность** - компонент в процессе перетаскивания

### Курсоры

- **👆 Рука** - можно взять компонент
- **✊ Захват** - компонент перетаскивается
- **🚫 Запрет** - компонент несовместим (нельзя перетащить)

## Автоматическая проверка совместимости

Система автоматически проверяет:
- Соответствие типа компонента и слота
- Совместимость сокета процессора и материнской платы
- Тип оперативной памяти (DDR4/DDR5)
- Длину видеокарты и размер корпуса
- Наличие свободных слотов для накопителей
- Форм-фактор материнской платы и корпуса

## Удаление компонентов

- Наведите на установленный компонент
- Появится кнопка "×" в правом верхнем углу
- Нажмите на неё для удаления компонента

## Порядок установки (рекомендуемый)

1. **Корпус** - выберите размер и форм-фактор
2. **Материнская плата** - основа системы
3. **Процессор** - совместимый с сокетом материнской платы
4. **Оперативная память** - правильный тип (DDR4/DDR5)
5. **Видеокарта** (опционально) - для игр и графики
6. **Накопители** - SSD для системы, HDD для данных
7. **Блок питания** - достаточной мощности

## Горячие клавиши

- Нет необходимости в горячих клавишах - просто перетаскивайте!

## Советы

✅ **Совет 1:** Начните с корпуса и материнской платы - это основа сборки

✅ **Совет 2:** Обращайте внимание на подсветку слотов - она подскажет, куда можно установить компонент

✅ **Совет 3:** Несовместимые компоненты отображаются полупрозрачными и их нельзя перетащить

✅ **Совет 4:** Читайте сообщения об ошибках - они содержат полезные рекомендации

## Технические детали

Функционал реализован с использованием:
- HTML5 Drag and Drop API
- React hooks (useState) для управления состоянием
- TypeScript для типобезопасности
- CSS анимации для визуальных эффектов
