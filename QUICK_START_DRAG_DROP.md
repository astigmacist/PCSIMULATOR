## 🎮 Первая сборка за 60 секунд

### Шаг 1: Корпус (5 сек)
```
1. Откройте категорию "🏠 Корпус"
2. Перетащите любой корпус в большую зону справа
✅ Корпус установлен!
```

### Шаг 2: Материнская плата (10 сек)
```
1. Откройте "🔌 Материнская плата"
2. Перетащите любую плату в центральный слот
✅ Основа готова!
```

### Шаг 3: Процессор (10 сек)
```
1. Откройте "🧠 Процессор"
2. Найдите процессор с ТАКИМ ЖЕ сокетом
   (например: AM5 → AM5)
3. Перетащите в маленький слот внутри материнской платы
✅ Мозг системы на месте!
```

### Шаг 4: Оперативная память (10 сек)
```
1. Откройте "💾 Оперативная память"
2. Найдите память ТОГО ЖЕ типа, что и плата
   (например: DDR5 → DDR5)
3. Перетащите в любой из 4 слотов RAM
✅ Память установлена!
```

### Шаг 5: Накопитель (10 сек)
```
1. Откройте "💿 Накопители"
2. Выберите любой SSD
3. Перетащите в слот "M.2/NVMe"
✅ Место для данных готово!
```

### Шаг 6: Блок питания (10 сек)
```
1. Откройте "⚡ Блок питания"
2. Выберите БП с достаточной мощностью
3. Перетащите в нижний правый слот
✅ Энергия есть!
```

### Шаг 7 (опционально): Видеокарта (5 сек)
```
1. Откройте "🎮 Видеокарта"
2. Выберите любую видеокарту
3. Перетащите в слот "Видеокарта"
✅ Игровая мощь добавлена!
```

## 🎯 Подсказки по совместимости

### ✅ Что нужно проверять:

**Процессор ↔ Материнская плата**
- Сокет должен совпадать
- AM5 = AM5, LGA1700 = LGA1700

**Память ↔ Материнская плата**
- Тип памяти должен совпадать
- DDR4 = DDR4, DDR5 = DDR5

**Видеокарта ↔ Корпус**
- Длина видеокарты ≤ максимальной длины корпуса

**Блок питания ↔ Система**
- Мощность БП ≥ суммарному потреблению

## 🖱️ Горячие подсказки

### Курсоры
- 👆 **Открытая ладонь** = можно взять
- ✊ **Сжатая ладонь** = сейчас тащите
- 🚫 **Запрет** = несовместимо

### Цвета
- 🟢 **Зеленый** = установлено
- 🟡 **Желтый** = сюда можно бросить
- 🔴 **Красный** = несовместимо
- ⚪ **Серый** = пусто

### Кнопки
- **×** = удалить компонент (появляется при hover)
- **▼** = развернуть категорию
- **▶** = свернуть категорию

## ❗ Типичные ошибки

### "Сначала установите материнскую плату!"
**Решение:** Процессор, RAM, GPU и накопители можно устанавливать только после материнской платы

### "Несовместимый сокет"
**Решение:** Выберите процессор с таким же сокетом, как у материнской платы
- Смотрите на строку "Сокет: AM5" в характеристиках

### "Неправильный тип памяти"
**Решение:** Выберите память того же типа, что поддерживает материнская плата
- Смотрите на строку "RAM: DDR5" в характеристиках платы

### "Видеокарта слишком длинная"
**Решение:** Выберите корпус побольше или видеокарту покороче
- Длина GPU ≤ Макс. длина корпуса

## 💡 Профессиональные советы

### Совет 1: Порядок имеет значение
```
Рекомендуемый порядок:
1. Корпус
2. Материнская плата
3. Процессор
4. Оперативная память
5. Накопители
6. Блок питания
7. Видеокарта (опционально)
```

### Совет 2: Смотрите на подсветку
- Если слот **желтый и пульсирует** → можно отпускать
- Если слот **не реагирует** → не тот тип компонента

### Совет 3: Читайте характеристики
- Наведите на компонент
- Все важные характеристики видны сразу
- Сравнивайте перед установкой

### Совет 4: Не бойтесь экспериментировать
- Система не даст установить несовместимый компонент
- Всегда можно удалить и попробовать снова
- Кнопка "🔄 Сбросить" начнет сборку заново
