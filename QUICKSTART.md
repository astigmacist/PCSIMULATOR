# 🚀 Быстрый старт

## Запуск проекта за 3 шага:

### 1️⃣ Откройте терминал

**Windows:**
- Нажмите `Win + R`
- Введите `cmd` и нажмите Enter
- Перейдите в папку проекта:
```bash
cd "C:\путь\к\папке\PC simulator"
```

**macOS:**
- Нажмите `Cmd + Пробел`
- Введите `Terminal` и нажмите Enter
- Перейдите в папку проекта:
```bash
cd "/Users/ваше_имя/Desktop/PC simulator"
```

### 2️⃣ Установите зависимости

```bash
npm install
```

⏱️ Это займет 2-3 минуты при первом запуске.

### 3️⃣ Запустите проект

```bash
npm run dev
```

🎉 Готово! Откройте браузер и перейдите по адресу:
```
http://localhost:5173
```

## ❓ Решение проблем

### Ошибка "npm не найден"

Установите Node.js:
- Скачайте с https://nodejs.org/
- Установите версию LTS (рекомендуется)
- Перезапустите терминал

### Порт занят

Если порт 5173 занят, Vite автоматически выберет другой (например, 5174).
Проверьте вывод в терминале.

### Очистить кэш

```bash
rm -rf node_modules package-lock.json
npm install
```

## 🎮 Как играть

1. **Выберите корпус** - начните с выбора корпуса
2. **Установите материнскую плату** - основа системы
3. **Добавьте процессор** - проверьте совместимость сокета
4. **Установите RAM** - минимум 8GB
5. **Добавьте видеокарту** (опционально)
6. **Выберите накопитель** - SSD или HDD
7. **Установите блок питания** - достаточной мощности
