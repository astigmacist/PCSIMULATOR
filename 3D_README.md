# 🎮 3D РЕЖИМ ДОБАВЛЕН!

## 🎉 Что изменилось?

Теперь симулятор поддерживает **два режима визуализации**:

### 📱 2D Режим (по умолчанию)
- Классическая схема с слотами
- Быстрая работа
- Простой интерфейс
- Подходит для старых компьютеров

### 🎮 3D Режим (НОВОЕ!)
- **Реалистичные 3D модели** всех компонентов
- **Интерактивный корпус ПК** со стеклянными панелями
- **RGB подсветка** и эффекты освещения
- **Управление камерой** (вращение, масштаб, перемещение)
- **Анимации** и динамические эффекты
- **Как в настоящей игре!** 🕹️

## 🚀 Как использовать:

### Шаг 1: Установите зависимости
```bash
npm install
```
Теперь будут установлены дополнительные 3D библиотеки:
- `@react-three/fiber` - React для Three.js
- `@react-three/drei` - Вспомогательные компоненты
- `three` - 3D движок

### Шаг 2: Запустите проект
```bash
npm run dev
```

### Шаг 3: Попробуйте 3D режим!
1. Откройте браузер: `http://localhost:5173`
2. Нажмите кнопку **"🎮 3D Режим"** в шапке
3. Используйте мышь для навигации:
   - **ЛКМ + движение** = вращение
   - **ПКМ + движение** = перемещение
   - **Колесико** = масштаб

## 🎨 Что вы увидите:

### В 3D режиме:
- 🏠 **Корпус ПК** с прозрачными стеклянными стенками
- 🟣 **RGB подсветка** по всему корпусу
- 🔌 **Все установленные компоненты** в реальном масштабе
- 📝 **Названия** компонентов парят над ними
- 🌀 **Вентиляционные отверстия** на корпусе
- 📊 **Сетка пола** для пространственной ориентации
- 💡 **Динамическое освещение** с нескольких источников

### Каждый компонент детализирован:
- **Материнская плата**: видны слоты, чипсет, разъемы
- **Процессор**: контакты, крышка, детали
- **RAM**: чипы памяти, радиатор
- **Видеокарта**: два кулера, радиатор, бэкплейт
- **БП**: вентилятор, решетка, разъемы
- **Накопители**: этикетки, разъемы
