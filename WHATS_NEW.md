# 🎉 Что нового? - Обновление Drag & Drop

## 🖱️ Главное нововведение: Drag & Drop!

Теперь вы можете **перетаскивать** комплектующие прямо в слоты, как в реальной жизни!

---

## 🚀 Как это работает?

### 3 простых шага:

```
1️⃣ Откройте категорию (нажмите на заголовок)
        ↓
2️⃣ Перетащите компонент в нужный слот →
        ↓
3️⃣ Отпустите - готово! ✅
```

---

## 💡 Визуальные подсказки

При работе вы увидите:

🟢 **Зеленая рамка** 
   → Компонент успешно установлен

🟡 **Желтая пульсация** 
   → Слот готов принять компонент (можно отпускать!)

🔴 **Красная рамка** 
   → Компонент несовместим (проверьте характеристики)

⚪ **Полупрозрачность** 
   → Компонент в процессе перетаскивания

🖱️ **"Перетащите →"** 
   → Появляется при наведении на компонент

---

## 📍 Где что устанавливать?

| Компонент | Куда тащить |
|-----------|-------------|
| 🏠 Корпус | Большая зона вверху справа |
| 🔌 Материнская плата | Центральный слот |
| 🧠 Процессор | Маленький слот внутри материнской платы |
| 💾 RAM | Слоты RAM 1-4 |
| 🎮 Видеокарта | Слот "Видеокарта" |
| 💿 Накопители | M.2/NVMe и SATA слоты |
| ⚡ Блок питания | Нижний правый слот |

## 🎯 Советы по использованию

💡 **Совет 1:** Начните с корпуса - перетащите его в специальную зону

💡 **Совет 2:** Следите за подсветкой слотов - она покажет, куда можно установить компонент

💡 **Совет 3:** Если компонент несовместим, система скажет почему и что с этим делать

💡 **Совет 4:** Можно удалить компонент, нажав на "×" при наведении

---

## 🎮 Попробуйте прямо сейчас!

1. Запустите проект: `npm run dev`
2. Откройте категорию "Корпус"
3. Перетащите любой корпус в зону справа
4. Продолжайте собирать ПК!
