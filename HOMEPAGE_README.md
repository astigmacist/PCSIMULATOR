# 🏠 Главная страница (Landing Page)

## ✨ Что добавлено

### Новая структура проекта:
```
/                    - Главная страница (landing)
/simulator           - Сам симулятор сборки ПК
```

## 📄 Секции главной страницы

### 1. **Hero Section** (Главный экран)
- Большой заголовок с призывом к действию
- Описание симулятора
- Две кнопки:
  - "🚀 Начать сборку" - ведет к симулятору
  - "📚 Узнать больше" - скролл к секции возможностей
- Статистика проекта (35+ компонентов, 10+ проверок, 2 режима)
- Анимированные карточки с компонентами (CPU, GPU, RAM)

### 2. **Features Section** (Возможности)
6 карточек с основными функциями:
- 🖱️ Drag & Drop интерфейс
- ✅ Проверка совместимости
- 🎮 3D Визуализация
- 📚 Обучающий режим
- 💰 Расчет стоимости
- 🔌 База комплектующих

### 3. **How It Works** (Как это работает)
3 шага процесса сборки:
1. Выберите компоненты
2. Перетащите в слоты
3. Проверьте результат

### 4. **CTA Section** (Призыв к действию)
- Финальный призыв начать сборку
- Большая кнопка "🚀 Запустить симулятор"

### 5. **Footer** (Подвал)
- Информация о проекте
- Список возможностей
- Используемые технологии
- Copyright

## 📱 Адаптивность

Главная страница полностью адаптивна:

### Desktop (> 1024px):
- Hero секция в 2 колонки
- Features grid в 3 колонки
- Все элементы в полную ширину

### Tablet (768px - 1024px):
- Hero секция вертикально
- Features grid в 1 колонку
- Оптимизированные размеры

### Mobile (< 768px):
- Вертикальное расположение
- Кнопки на всю ширину
- Уменьшенные шрифты
- Упрощенные анимации

## 🚀 Как использовать

### Запуск проекта:
```bash
npm run dev
```

### Навигация:
- Открывается главная страница (/)
- Кнопка "Начать сборку" → переход к симулятору
- В симуляторе кнопка "← Главная" → возврат на главную

## 🎯 Преимущества новой структуры

1. **Профессиональный вид**
