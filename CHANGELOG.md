# 📝 История изменений

## Версия 2.0 - Drag & Drop Update (Январь 2026)

### ✨ Новые возможности

#### 🖱️ Drag & Drop интерфейс
- **Перетаскивание компонентов**: Теперь можно перетаскивать комплектующие из левой панели в слоты справа
- **Визуальная обратная связь**: 
  - Компоненты становятся полупрозрачными при захвате
  - Слоты подсвечиваются желтым с пульсацией при наведении
  - Курсор меняется (рука → захват)
- **Автоматическая валидация**: Система проверяет совместимость при перетаскивании
- **Индикатор перетаскивания**: Подсказка "🖱️ Перетащите →" появляется при наведении

#### 🎨 Улучшения интерфейса
- **Улучшенное отображение списка**: 
  - Исправлено отображение всех компонентов
  - Добавлен скроллинг для длинных списков
  - Оптимизирована высота категорий
- **Аккордеон для категорий**: 
  - Плавное раскрытие/закрытие категорий
  - Улучшенная анимация
  - Sticky заголовок при скролле
- **Зона для корпуса**: 
  - Специальная drop-zone для корпуса вверху правой панели
  - Информационная панель с установленным корпусом
  - Возможность быстрого удаления

### 🔧 Технические улучшения

#### Компоненты
- **ComponentSelection.tsx**: 
  - Добавлены drag handlers (onDragStart, onDragEnd)
  - Управление состоянием перетаскивания
  - Индикаторы для draggable элементов
  
- **PCCaseView.tsx**: 
  - Добавлены drop zones для всех слотов
  - Обработчики drag events (onDragOver, onDragLeave, onDrop)
  - Валидация при drop
  - Специальная зона для корпуса

#### Стили (App.css)
- Новые CSS классы:
  - `.drag-hint` - индикатор возможности перетаскивания
  - `.drag-over` - подсветка активного слота
  - `.case-drop-zone` - зона для корпуса
  - `.case-info-bar` - информация об установленном корпусе
- Анимации:
  - `pulse` - пульсация слотов при drag over
  - Плавные переходы для всех состояний
- Улучшенные курсоры:
  - `cursor: grab` для draggable элементов
  - `cursor: grabbing` при активном перетаскивании
