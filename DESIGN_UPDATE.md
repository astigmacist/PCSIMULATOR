## ✨ Анимации и эффекты

### Float (Плавание):
```css
@keyframes float {
  0%, 100% {
    transform: translateY(0px) rotate(0deg);
  }
  50% {
    transform: translateY(-20px) rotate(3deg);
  }
}
```
- Длительность: 6 секунд
- Easing: ease-in-out
- Бесконечный цикл

### Pulse (Пульсация):
```css
@keyframes pulse {
  0%, 100% {
    transform: scale(1);
    opacity: 1;
  }
  50% {
    transform: scale(1.05);
    opacity: 0.9;
  }
}
```
- Используется для HOT бейджа
- Длительность: 2 секунды

### Hover эффекты:

**Плавающие карточки:**
- Подъем: `translateY(-10px)`
- Тень: `0 20px 60px rgba(59, 130, 246, 0.4)`
- Рамка: `rgba(59, 130, 246, 0.6)`
- Transition: `0.4s cubic-bezier(0.4, 0, 0.2, 1)`

**Showcase карточки:**
- Подъем: `translateY(-8px)`
- Иконка: `scale(1.15) rotate(5deg)`
- Тень: `0 16px 48px rgba(59, 130, 246, 0.25)`
- Transition: `0.4s cubic-bezier(0.4, 0, 0.2, 1)`

## 💡 Советы по использованию

1. **Не бойтесь экспериментировать с градиентами**
   - Меняйте цвета в CSS классах `.gradient-*`
   - Используйте генераторы градиентов

2. **SVG легко кастомизировать**
   - Меняйте `stroke-width` для толщины линий
   - Добавляйте/убирайте элементы
   - Используйте `currentColor` для наследования

3. **Анимации должны быть деликатными**
   - Не перегружайте эффектами
   - Используйте `ease-in-out` для плавности
   - Тестируйте на слабых устройствах

4. **Поддерживайте единый стиль**
   - Одинаковые border-radius
   - Одинаковые тени
   - Согласованная типографика
