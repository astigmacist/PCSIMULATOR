# 🎮 Возможности Симулятора Сборки ПК

## ✨ Основные возможности

### 1. **Интерактивная визуализация** 🖥️
- Реалистичное отображение корпуса ПК
- Визуальные слоты для каждого компонента
- Подсветка установленных компонентов
- Возможность удаления компонентов одним кликом

### 2. **Автоматическая проверка совместимости** ✅
Система проверяет:
- ✓ Совместимость сокета процессора и материнской платы
- ✓ TDP процессора и возможности материнской платы
- ✓ Тип оперативной памяти (DDR3/DDR4/DDR5)
- ✓ Количество и объем слотов RAM
- ✓ Размер видеокарты относительно корпуса
- ✓ Количество PCI слотов
- ✓ Доступность SATA и M.2 слотов
- ✓ Мощность блока питания
- ✓ Форм-фактор материнской платы и корпуса

### 3. **База данных комплектующих** 📦
#### Материнские платы (5 моделей)
- AM4, AM5, LGA1700 сокеты
- DDR4 и DDR5 поддержка
- Различные форм-факторы (ATX, Micro-ATX)

#### Процессоры (6 моделей)
- AMD Ryzen серии 5000 и 7000
- Intel Core 12-го поколения
- От 6 до 16 ядер

#### Оперативная память (5 моделей)
- DDR4 и DDR5
- От 8GB до 64GB
- Частоты от 2666MHz до 6000MHz

#### Видеокарты (5 моделей)
- NVIDIA RTX 3060 Ti - RTX 4090
- AMD Radeon RX 6600 - RX 7800 XT
- Различное энергопотребление

#### Блоки питания (5 моделей)
- От 500W до 1200W
- Сертификаты 80+ Bronze/Gold/Platinum
- Модульные и немодульные

#### Накопители (5 моделей)
- NVMe SSD (высокая скорость)
- SATA SSD
- HDD для больших объемов

#### Корпуса (4 модели)
- Различные размеры
- Поддержка разных форм-факторов
- Ограничения по длине видеокарты

### 4. **Умные подсказки** 💡
При попытке установить несовместимый компонент:
- ❌ Описание проблемы
- 📋 Причина несовместимости
- 💡 Рекомендации по исправлению
- 📝 Список совместимых альтернатив

### 5. **Система расчетов** 📊
- **Стоимость**: Автоматический расчет общей стоимости сборки
- **Энергопотребление**: Расчет потребления всех компонентов
- **Проверка мощности**: Предупреждение о недостатке мощности БП

### 6. **Обучающий режим** 🎓
4 готовых задания:
1. **💼 Офисный ПК** (бюджет: 250,000₸)
   - Для работы с документами
   - Без видеокарты
   - Минимальные требования

2. **🎮 Бюджетный игровой ПК** (бюджет: 400,000₸)
   - Для популярных игр
   - С видеокартой
   - Оптимальное соотношение цена/качество

3. **🚀 Мощный игровой ПК** (бюджет: 700,000₸)
   - Для AAA-игр в высоком качестве
   - Топовые компоненты
   - DDR5 память

4. **💻 Рабочая станция** (бюджет: 800,000₸)
   - Для рендеринга и 3D
   - Многоядерный процессор
   - Большой объем RAM

Каждое задание включает:
- 📋 Четкие требования
- 💡 Подсказки (можно скрыть)
- ✅ Автоматическая проверка выполнения
- 📊 Отслеживание прогресса

### 7. **Игровой интерфейс** 🎨
- **Современный дизайн** с градиентами
- **Плавные анимации** при наведении
- **Подсветка** совместимых/несовместимых компонентов
- **Визуальная обратная связь** на каждое действие
- **Эффекты** при установке компонентов

### 8. **Панель статуса** 📈
Отображает:
- ✅/❌ Установлены ли все компоненты
- 💾 Общий объем RAM
- 💿 Общий объем накопителей
- ⚡ Текущее энергопотребление
- 💰 Общая стоимость сборки
- 🎉 Уведомление о завершении сборки

### 9. **Панель совместимости** 🔍
Показывает:
- ❌ **Ошибки** с подробным описанием
- ⚠️ **Предупреждения** о недостающих компонентах
- 💡 **Рекомендации** по исправлению

### 10. **Удобное управление** 🎯
- **Клик для установки** компонента
- **Кнопка удаления** на каждом установленном компоненте
- **Кнопка сброса** для начала новой сборки
- **Кнопка помощи** с инструкциями
- **Кнопка заданий** для обучающего режима
