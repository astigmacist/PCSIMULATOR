## 🔄 Как заменить изображения

Если хотите использовать другие изображения:

### Вариант 1: Другие фото с Unsplash
1. Перейдите на https://unsplash.com
2. Найдите нужное изображение
3. Скопируйте URL в формате: `https://images.unsplash.com/photo-ID?w=600&q=80`
4. Замените в `HomePage.tsx`

### Вариант 2: Локальные изображения
1. Создайте папку `public/images/`
2. Добавьте туда свои изображения
3. Измените путь: `src="/images/your-image.jpg"`

### Вариант 3: CDN производителей
Используйте официальные изображения:
```tsx
// Пример для AMD
src="https://www.amd.com/system/files/images/ryzen9-7950x.jpg"

// Пример для NVIDIA
src="https://nvidia.com/content/dam/rtx-4090.jpg"
```

## 🚀 Производительность

### Оптимизации:
- ✅ Параметр `w=600` - ширина ограничена
- ✅ Параметр `q=80` - качество сжатия 80%
- ✅ Lazy loading (автоматически в браузере)
- ✅ CDN Unsplash - быстрая доставка
- ✅ WebP формат (современные браузеры)

### Средний размер:
- Одно изображение: ~50-100KB
- Всего изображений: 7 штук
- Общий размер: ~400-700KB
- Время загрузки: < 2 секунд

## 💡 Советы

1. **Используйте CDN** - быстрее локальных файлов
2. **Оптимизируйте размер** - параметр `w=` в URL
3. **Качество 80%** - хороший баланс размер/качество
4. **Lazy loading** - изображения загружаются по мере прокрутки
5. **Alt текст** - всегда добавляйте для SEO и доступности
