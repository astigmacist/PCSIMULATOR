# 🖥️ Симулятор Сборки ПК

Интерактивный веб-симулятор для обучения сборке персонального компьютера с проверкой совместимости комплектующих.

![PC Simulator](https://img.shields.io/badge/React-18.2-blue) ![TypeScript](https://img.shields.io/badge/TypeScript-5.2-blue) ![Vite](https://img.shields.io/badge/Vite-5.0-purple)

## 🎯 Возможности

- ✅ **Drag & Drop интерфейс** - перетаскивайте компоненты в нужные слоты
- ✅ **Интерактивная визуализация** корпуса ПК с установленными компонентами
- ✅ **Автоматическая проверка совместимости** всех комплектующих
- ✅ **База данных комплектующих** с реальными характеристиками
- ✅ **Подробные подсказки** при ошибках установки
- ✅ **Расчет стоимости** и энергопотребления системы
- ✅ **Красивый игровой интерфейс** с анимациями и плавными переходами

## 🚀 Быстрый старт

### Предварительные требования

- Node.js (версия 16 или выше)
- npm или yarn

### Установка

1. **Откройте терминал** в папке проекта

2. **Установите зависимости:**
   ```bash
   npm install
   ```

3. **Запустите проект:**
   ```bash
   npm run dev
   ```

4. **Откройте браузер** и перейдите по адресу:
   ```
   http://localhost:5173
   ```

## 📖 Как использовать

### 🖱️ Управление Drag & Drop

**Новинка!** Теперь вы можете перетаскивать компоненты прямо в слоты:

1. **Откройте категорию** - нажмите на заголовок нужной категории комплектующих
2. **Выберите компонент** - наведите курсор на нужный компонент (появится подсказка "🖱️ Перетащите →")
3. **Перетащите** - зажмите левую кнопку мыши и перетащите компонент на нужный слот в правой части
4. **Отпустите** - когда слот подсветится желтым, отпустите кнопку мыши

**Визуальные подсказки:**
- 🟢 **Зеленая рамка** - компонент успешно установлен
- 🟡 **Желтая пульсация** - слот готов принять компонент
- 🔴 **Красная рамка** - компонент несовместим
- ⚪ **Полупрозрачность** - компонент в процессе перетаскивания

📖 Подробная инструкция по Drag & Drop: [DRAG_AND_DROP.md](./DRAG_AND_DROP.md)

### Пошаговая инструкция по сборке ПК:

1. **Выберите корпус** 🏠
   - Перетащите корпус в специальную зону вверху правой панели
   - Обратите внимание на поддерживаемые форм-факторы

2. **Установите материнскую плату** 🔌
   - Перетащите материнскую плату в большой центральный слот
   - Выберите нужный сокет (AM4, AM5, LGA1700)
   - Обратите внимание на тип поддерживаемой памяти (DDR4/DDR5)

3. **Добавьте процессор** 🧠
   - Перетащите процессор в маленький слот внутри материнской платы
   - Убедитесь, что сокет процессора совпадает с материнской платой
   - Проверьте TDP (тепловыделение)

4. **Установите оперативную память** 💾
   - Перетащите планки памяти в слоты RAM 1-4
   - Выберите память правильного типа (DDR4 или DDR5)
   - Можно установить до 4 планок памяти

5. **Добавьте видеокарту** 🎮 (опционально)
   - Перетащите видеокарту в соответствующий слот
   - Для игр и работы с графикой
   - Проверьте размер относительно корпуса

6. **Выберите накопитель** 💿
   - Перетащите накопители в слоты M.2/NVMe или SATA
   - SSD (NVMe/SATA) для быстрой работы
   - HDD для большого объема данных
   - Можно установить несколько накопителей

7. **Установите блок питания** ⚡
   - Перетащите БП в нижний правый слот
   - Убедитесь, что мощности хватает для всех компонентов
   - Система автоматически рассчитает необходимую мощность

### Проверка совместимости

Система автоматически проверяет:
- ✅ Совместимость сокета процессора и материнской платы
- ✅ Соответствие типа оперативной памяти
- ✅ Размер видеокарты относительно корпуса
- ✅ Достаточность мощности блока питания
- ✅ Наличие свободных слотов для накопителей

При несовместимости вы получите:
- ❌ Уведомление об ошибке
- 💡 Причину несовместимости
- 📝 Рекомендации по исправлению

## 🐛 Решение проблем

### Проект не запускается
```bash
# Удалите node_modules и переустановите зависимости
rm -rf node_modules package-lock.json
npm install
```

### Ошибки TypeScript
```bash
# Пересоберите проект
npm run build
```

### Порт занят
Vite автоматически выберет другой порт. Проверьте вывод в терминале.
