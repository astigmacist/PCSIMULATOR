# Папка для 3D моделей

## Структура папок

Поместите ваши .obj файлы в эту папку по следующей структуре:

```
public/models/
├── cases/           # Модели корпусов ПК
│   ├── nzxt-h510.obj
│   ├── nzxt-h510.mtl  (опционально)
│   └── ...
├── motherboards/    # Модели материнских плат
├── cpu/            # Модели процессоров
├── gpu/            # Модели видеокарт
├── ram/            # Модели памяти
├── psu/            # Модели блоков питания
└── storage/        # Модели накопителей
```

## Форматы файлов

- `.obj` - основной файл модели (обязательно)
- `.mtl` - файл материалов (опционально, но рекомендуется)
- Текстуры (если есть) - положите рядом с .obj файлом

## Пример использования

После размещения файлов в этой папке, они будут доступны по URL:
- `/models/cases/nzxt-h510.obj`
- `/models/gpu/rtx-4090.obj`

## Рекомендации

1. Называйте файлы латиницей и без пробелов
2. Оптимизируйте модели (не более 50k полигонов)
3. Центрируйте модель в origin (0,0,0)
4. Используйте правильный масштаб (1 unit = 1 см)
