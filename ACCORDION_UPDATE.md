# 🎯 Аккордеон категорий добавлен!

## ✅ Что изменилось:

Теперь **7 категорий** компонентов работают как **аккордеон** (можно сворачивать/разворачивать)!

---

## 📦 7 Категорий компонентов:

1. 🏠 **Корпус** (Case)
2. 🔌 **Материнская плата** (Motherboard)
3. 🧠 **Процессор** (CPU)
4. 💾 **Оперативная память** (RAM)
5. 🎮 **Видеокарта** (GPU)
6. 💿 **Накопители** (Storage)
7. ⚡ **Блок питания** (PSU)

---

## ✨ Новые возможности:

### 1. **Сворачиваемые карточки**
- ✅ Каждая категория - отдельная карточка
- ✅ Клик на заголовок - разворачивает/сворачивает
- ✅ Иконка показывает состояние (▶ закрыто, ▼ открыто)

### 2. **Умный скроллинг**
- ✅ **Карточки категорий НЕ скроллятся** - остаются на месте
- ✅ **Внутри категории скроллится список** комплектующих (макс. 400px)
- ✅ Плавный скролл с красивым скроллбаром

### 3. **По умолчанию**
- ✅ Открыта только категория **"Корпус"**
- ✅ Все остальные свернуты
- ✅ Можно открыть несколько одновременно

### 4. **Красивый дизайн**
- 🎨 Фиолетовая подсветка
- ✨ Анимация раскрытия (slideDown)
- 💫 Hover эффекты
- 🌟 Иконки для каждой категории
- 📦 Компактный вид

---

## 🎮 Как использовать:

### Раскрыть категорию:
```
Клик на заголовок категории → категория раскрывается
```

### Свернуть категорию:
```
Еще один клик → категория сворачивается
```

### Скролл внутри:
```
Если компонентов много → скроллится только внутри карточки
Максимальная высота списка: 400px
```

## 🎨 Визуальные особенности:

### Заголовок категории:
- Иконка компонента (цветная SVG)
- Название категории
- Стрелка-индикатор (▶/▼)
- Фиолетовый фон при наведении

### Список компонентов:
- Максимум 400px высоты
- Кастомный скроллбар (фиолетовый)
- Анимация появления
- Сохраняет все карточки компонентов

### Эффекты:
- ✨ Плавное раскрытие (0.3s)
- 💫 Подсветка при hover
- 🌟 Тени для глубины
- 📦 Скругленные углы
