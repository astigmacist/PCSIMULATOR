## 🎨 Новые возможности в действии

### Drag & Drop:
```
1. Наведите на компонент
   └─ Появится "🖱️ Перетащите →"

2. Зажмите левую кнопку мыши
   └─ Компонент станет полупрозрачным
   └─ Курсор изменится на "захват"

3. Переместите на слот
   └─ Слот подсветится желтым
   └─ Начнется пульсация

4. Отпустите кнопку
   └─ Компонент установлен! ✅
   └─ Зеленая анимация успеха
```

### Визуальные подсказки:
- 🟢 Зеленая рамка = установлено
- 🟡 Желтая пульсация = готов принять
- 🔴 Красная рамка = несовместимо
- 👆 Открытая ладонь = можно взять
- ✊ Сжатая ладонь = тащите

## ✨ Особенности реализации

### HTML5 Drag & Drop API:
```typescript
// Источник (ComponentSelection)
onDragStart={(e) => handleDragStart(e, 'cpu', cpu)}
onDragEnd={handleDragEnd}
draggable={true}

// Получатель (PCCaseView)
onDragOver={(e) => handleDragOver(e, 'cpu')}
onDragLeave={handleDragLeave}
onDrop={(e) => handleDrop(e, 'cpu')}
```

### Автоматическая валидация:
```typescript
// Проверка типа компонента
if (componentType !== slotType) {
  alert('Неправильный слот!');
  return;
}

// Проверка совместимости
const error = checkCPUCompatibility(cpu, motherboard);
if (error) {
  alert(error.reason + '\n' + error.recommendations);
  return;
}

// Установка
installCPU(cpu);
```
